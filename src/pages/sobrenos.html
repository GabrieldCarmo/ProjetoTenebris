Sobre Tenebris

Tenebris nasceu de uma paixão compartilhada pelo cinema de terror e pela programação web. Este projeto foi desenvolvido como trabalho de conclusão da disciplina de Programação Web no ensino médio, unindo cinco estudantes determinados a criar algo memorável. Durante meses de desenvolvimento, cada membro da equipe contribuiu com suas habilidades únicas, transformando a ideia inicial em uma plataforma completa dedicada às franquias mais aterrorizantes do cinema. De reuniões após as aulas a noites de código, cada linha foi escrita com dedicação e paixão pelo que fazemos. O nome "Tenebris" vem do latim, significando "trevas" ou "escuridão" - uma homenagem perfeita ao gênero que tanto amamos. Nosso objetivo foi criar não apenas um catálogo de filmes, mas uma experiência imersiva que captura a essência do terror cinematográfico. Este projeto representa não apenas uma nota final, mas o início de nossa jornada como desenvolvedores. Cada desafio superado, cada bug corrigido e cada feature implementada nos ensinou lições valiosas sobre trabalho em equipe, persistência e criatividade.
